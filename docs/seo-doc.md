# Hano SEO & Organic Growth Implementation Specification

## Overview

### Mission

Position Hano as the leading hospitality discovery platform in Rwanda by becoming the primary destination for users searching for restaurants, cafés, bars, lounges, hotels, guest houses, apartments, attractions, events, and experiences.

The SEO strategy must be designed to:

* Drive high-intent traffic.
* Generate mobile app downloads.
* Generate business registrations.
* Capture hospitality-related searches across Rwanda.
* Establish Hano as the authoritative source for hospitality discovery.

# Technical SEO Requirements

## Metadata

Every page must include:

### Title Tag

Format:

Business Name | Category | Location | Hano

Examples:

* Boho Restaurant | Restaurant in Kigali | Hano
* Heaven Hotel | Hotel in Kigali | Hano

---

### Meta Description

Format:

Discover menus, reviews, ratings, photos, opening hours, and more for {Business Name} on Hano.

Maximum length:

150-160 characters.

---

### Open Graph

Required:

* og:title
* og:description
* og:image
* og:url
* og:type

---

### Twitter Cards

Required:

* summary_large_image

---

# Canonical URL Strategy

Every page must define a canonical URL.

Examples:

```html
<link rel="canonical" href="https://hano.rw/restaurants/boho-kigali" />
```

This prevents duplicate indexing.
